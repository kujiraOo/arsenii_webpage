games

Orange Shooter

This web game was my final project for Basics of Programming course

This is a 2D scrolling shooter. Conrol the jet with WASD keys, press SPACE to shoot. You also have three abilities: Bullet Time, Super Shield and Big Bang. To use them press 1 2 3 keys when the ability gauge is full.

JavaScript, CraftyJS engine

Car Drop

Car Drop is actually not a game. It's android relax app where you have some basic interaction with objects. The idea is just to enjoy an endless journey.

Just press the acceleration button. Touch stars to get upgrades for the car. And some of the objects are as well responsive, try to figure which ones ;)

Graphics by Aigul Nigmetova

Lua, Gideros SDK
